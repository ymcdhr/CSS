元素类型
元素类型：
1、块级元素：块级元素占据其父元素（容器）的整个空间
2、行内元素（如：< a >、 < button >）：一个行内元素只占据它对应标签的边框所包含的空间。
3、替代元素（如：< button >、< input >、< img >）：可替换元素通过属性或者标签来展示元素。内容展现不是由CSS来控制的，这些元素是一类 外观渲染独立于CSS的 外部对象。
4、非替代元素（如：< p >、< label > ）：其内容全部被浏览器展示

line-height的理解
1、按元素类型区别
（1）块级元素：指定子元素的高度
（2）非替代行内元素（如：文本等浏览器直接显示的元素）：其行高度
（3）行内元素（如：< a > 、 < button >）：有一些
2、line-height 的 number | length | percentage
3、number 为无单位的数字，例如：line-height：1.5
4、percentage 为参考font-size的百分比，例如：line-height：150%
5、number和percentage都可以表示倍数（不过，number继承的时候子元素继承倍数，percentage子元素继承百分比算出来的值）
line-heigh设置行内元素的高度，实现文字的垂直居中
对于块级元素，会继承父元素的line-height
非替代行内元素
line-height：1.5，继承的时候子元素继承数字，再自己算
line-height：150%，继承的时候子元素继承百分比算出来的值
